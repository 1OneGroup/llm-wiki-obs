# obs+llmwiki

Multi-department LLM-powered wiki with Obsidian as the UI.

## Stack

- **Wiki engine:** [LLM Wiki](https://github.com/lucasastorian/llmwiki) — Claude writes wiki pages from source documents
- **UI:** [Obsidian](https://obsidian.md) — reads markdown files natively
- **Agent harness:** Inspired by [Archon](https://github.com/coleam00/Archon) — YAML workflows, typed nodes

## Quick Start

1. Open this folder as an Obsidian vault
2. Start with `_vault/INDEX.md` as your dashboard
3. Drop documents into `[Department]/sources/`
4. Run the wiki ingestion workflow
