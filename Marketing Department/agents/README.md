# Agents

Specialized AI agents for marketing operations.

| Agent | File | Role |
|---|---|---|
| Social Media Manager | `social-media-manager.md` | Content planning, copywriting, Postiz publishing |
| Graphic Designer | `graphic-designer.md` | Branded image generation via Nano Banana MCP |
| Telecalling Manager | `telecalling-manager.md` | Gemini Live AI voice call campaigns |

## Usage
Invoke agents from Claude Code by referencing the agent file as context.
Each agent has a specific role — do not mix responsibilities.
