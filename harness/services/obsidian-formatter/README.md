# Obsidian Formatter

Converts Claude's wiki page output into proper Obsidian markdown.

## Transformations
- Adds YAML frontmatter (title, tags, date_created, sources)
- Converts `[Page Name](/path)` links to `[[page-name]]` wikilinks
- Wraps key rules in `> [!note]` callout blocks
- Adds cross-links to related pages
